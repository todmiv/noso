# Ассоциация «НОСО» - Предложение о сотрудничестве

Это проект содержит лендинг-страницу для Ассоциации «Нижегородское объединение строительных организаций» (Ассоциация «НОСО»). Страница предлагает потенциальным партнерам вступить в саморегулируемую организацию (СРО) в строительной сфере.

## Описание проекта

Проект представляет собой одностраничный сайт (landing page) на HTML и CSS, который предоставляет информацию о преимуществах членства в Ассоциации «НОСО», условиях вступления, членских взносах и контактных данных.

### Основные разделы:
- **Заголовок**: Логотип и контактная информация ассоциации.
- **Приветствие**: Призыв к сотрудничеству с ключевыми преимуществами.
- **Преимущества**: Детальное описание выгод членства (скидки, отсутствие сборов, страховка и т.д.).
- **Членские взносы**: Таблица с уровнями ответственности и соответствующими платежами.
- **Компенсационный фонд**: Информация о обязательных платежах в соответствии с законодательством.
- **О нас**: Краткая история и текущий статус организации.
- **Контакты**: Телефоны, email и ссылка на официальный сайт.

## Технологии

- **HTML5**: Структура страницы.
- **CSS3**: Стилизация и адаптивный дизайн.

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/todmiv/noso.git
   ```

2. Откройте файл `index.html` в любом современном веб-браузере.

## Структура файлов

- `index.html` - Основная страница сайта.
- `LICENSE` - Лицензия проекта.
- `logo.png` - Логотип ассоциации (встроен в HTML как base64).
